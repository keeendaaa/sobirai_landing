# sobirAI — лендинг AI‑платформы

Одностраничный сайт для продукта sobirAI. Проект собран на Vite + React.

## Быстрый старт

Установка зависимостей:
```bash
npm install
```

Запуск в режиме разработки:
```bash
npm run dev
```

Приложение будет доступно по адресу `http://localhost:5173`.

## Сборка и предпросмотр

Сборка для продакшена:
```bash
npm run build
```

Файлы сборки появятся в папке `dist`.

Просмотр продакшен‑сборки локально:
```bash
npm run preview
```

## Структура проекта

```
├── src/
│   ├── App.jsx          # Главный компонент страницы
│   ├── main.jsx         # Точка входа
│   └── index.css        # Основные стили
├── public/
│   └── fonts/           # Локальные шрифты
├── assets/              # Изображения и ресурсы
├── index.html           # HTML шаблон
├── vite.config.js       # Конфигурация Vite
└── package.json         # Скрипты и зависимости
```

## Шрифты

Проект использует локальные шрифты, которые лежат в `public/fonts`.
Пути и подключения находятся в `src/index.css` и `index.html` (preload).

## Команды

- `npm run dev` — запуск разработки
- `npm run build` — сборка
- `npm run preview` — предпросмотр сборки
